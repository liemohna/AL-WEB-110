Lists

Unordered lists; use an h3 tag

Lists are very commonly used. Let's try to create an unordered nested list.

  • item 1
  • item 2
  • item 3
    • sub item 3.1
    • sub item 3.2

  • item 4
  • item 5

Ordered lists

Unordered lists are very common, but a second commonly used list type is an ordered list. Let's create one of those and nest it as well.

  1. item 1
  2. item 2
  3. item 3
    1. sub item 3a
    2. sub item 3b
    3. sub item 3c
  4. item 4
  5. item 5
